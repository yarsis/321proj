This file provides information on the controls, gameplay, and features of Metroidvania. It is recommended to read through
this document before playing. Metroidvania is a project completed by Caden, Addie, Rachel, and Henry for CS 321.

			    CONTROLS
=================================================================

KEYBOARD CONTROLS
+++++++++++++++++
'a' - Pressing the 'a' key will move the player in the left direction.
'd' - Pressing the 'd' key will move the player in the right direction.
'w' - Pressing the 'w' key will allow the player to jump. Note this can only be done while on terrain.
'<-' - Pressing the left arrow key will move the player in the left direction.
'->' - Pressing the right arrow key will move the player in the right direction.
'^' - Pressing the up arrow key will allow the player to jump. Note this can only be done while on terrain.
spacebar - Pressing the spacebar will allow the player to jump. Note this can only be done while on terrain.
esc - Pressing the escape key while in the game will open up the pause screen. From here the player can change settings, continue, or exit the game. 
Pressing the escape key while in a menu will return to the game.

MOUSE CONTROLS
+++++++++++++++++
left mouse click - A left mouse click will fire a projectile, or select an option in the menu.
mouse cursor - The mouse cursor is used to aim the projectile. It will be fired in the cardinal direction (up, left, or right) relative to the cursor's
location to the player. 

			    GAMEPLAY
=================================================================

POINTS
++++++++++++++++
The player spawns with 9 points to begin with. This score is displayed in the top right of the window. One point will be subtracted for every projectile
the player decides to shoot. Likewise, 10 points will be gained for successfully shooting an enemy. Carefully navigate the terrain and take out enemies while
avoiding bottomless pits to achieve a high score. 

HEALTH
++++++++++++++++
The player spawns with 100 health to begin with. Taking damage from enemies will result in a losing half your health. It is important to naviagte the stages 
carefully to avoid getting hit with enemy projectiles. Falling out of the map will result in instant death.

GAME OVER
++++++++++++++++
Upon reaching 0 health or falling out of the map, the game over screen is presented where the player can either quit the game or save their high score.

COLOR CHANGE
++++++++++++++++
After pressing escape to enter the pause menu, the player has the option to enter the "change colors" panel. From here, the user can cycle through background, enemy, 
and their own color in game. Press escape to return from this menu back to the game to see your new colors in action!
